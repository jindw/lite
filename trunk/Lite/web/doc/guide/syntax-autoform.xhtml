功能支持

  • select的自动选中

    在option 有selected 属性的时候，不做自动填充。

    当select 没有multiple属性的时候，当且仅当name所对应的值与当前option对应的值渲染的字符串一致是，设置selected="selected"

    当select 有multiple属性的时候，循环name所在的值，如果有与当前option value 字符串相等，则设置selected="selected"

  • textarea的自动填充

    当textarea 没有内容的时候，自动设置内容为：\${<INPUT_NAME>}

  • 普通input的自动填充

    这个比较简单，当autoform开启状态下，且input 没有指定value属性，且type!= image||radio||checkbox时，value自动设置为\${<INPUT_NAME>}

  • radio/checkbox 的自动勾选

    实现方式与select option 的mutiple 和非mutiple相似。
