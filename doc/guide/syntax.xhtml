Lite基本语法，命名为LiteXML，是一个基于XML的基本模板语法聚合，包括最基本的模板功能。

LiteXML主要功能有

  • 输出语法

    表达式输出默认是自动编码的。其中属性的输出还隐含一个条件控制。

    详情见：

  • 条件控制

    条件控制包含 c:if c:else; c:choose c:when 两种书写风格

    详情见：、

  • 循环控制

    for 循环可以同时实现对 数组，对象(Map),和数字三种实体循环。

    详情见：

  • 函数/变量申明

    通过 c:var 语法申明变量、 c:def 语法申明函数。

    详情见：、

  • 模板继承

    通过 c:extension 语法实现模板继承,他不仅可以实现页面的继承，还能实现页面元素的继承。

    详情见：

  • 静态包含

    通过 c:include语法实现模板包含（不能包含不完整的节点片段），而且配合xpath，可以实现页面片段的包含。

    详情见：

  • 前端模板

    同时支持前端模板是lite模板引擎的一大亮点。

    详情见：

  • 自己扩展

    如果你觉得这些还不够用，那你就自己扩展吧。
