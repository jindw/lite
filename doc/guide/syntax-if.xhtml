语法功能

  • 实现条件判断

    无论你是什么编程语言，条件判断的规则都是JavaScript规则。

    空字符串"",false,0,null是false；其他都是真值（{},[]等，都是真值）。

    示例 您好： ${displayName} ${loginName} 匆匆过客； 今天是： 休息 工作日 ]]>

    小技巧1： JavaScript 与或运算（JavaScript与或技巧）和三目运算符号，也可以解决上述问题。语法如下： 您好：${displayName||loginName||'匆匆过客'}；今天是：${day>5?'休息':'工作'}日 ]]>
